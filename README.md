# chat-1.0

## О проекте

Чат для переписки, выполнен с целью изучения Firebase. Регистрация пользователя с помощью Firebase/Authentication. Сохрание сообщений в чате с помощью Firebase/Firestore.

## Стек

- firebase
- react JS
- react-firebase-hooks
- material-ui
